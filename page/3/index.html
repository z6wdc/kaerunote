Python collections defaultdict

Posted on 2021-07-06 Edited on 2021-07-20

defaultdict

可以用來生成 dictionary

而且還可以設定 value 的預設值

from collections import defaultdict

d = defaultdict(int)

上面的例子代入了 int

所以 key 預設的值會使用 int()

而 int() 的結果就會是 0

lambda

當然也可以不用預設值一定要是 0

用 lambda 可以更靈活的設定預設值

from collections import defaultdict

d = defaultdict(lambda: 3)

isnull()

可以用 isnull() 來找出 NaN 的值

也可以寫成 isna()

df.isnull()

也可以和 any() 跟 sum() 搭配使用

sum()

df.isnull().sum()

df.isnull().sum(axis=1)

df.isnull().any()

query = df.isnull().any(axis=1)
df[query].head()

pandas groupby

Posted on 2021-06-30 Edited on 2021-07-20

groupby

groupby 用來對資料做集結

df.groupby('column1').sum()

也可對複數個 column 做集結

df.groupby(['column1','column2']).sum()

集結後也可以指定對哪個 column 做處理

df.groupby('column1').sum()['column2']

也可以指定複數個 column

df.groupby('column1').sum()[['column2','column3']]

不指定的話，就是對 column1 以外的 column 做處理

關於處理，也可以同時指定多個

例如同時使用 count sum max min

df.groupby('column1').agg(['count','sum','max','min'])['column2']

pandas head tail

head

tail

pandas 可以利用 head 跟 tail 來取出前幾筆跟後幾筆的資料

如果直接使用，預設會取出前 5 筆跟後 5 筆的資料

df.head()
df.tail()

也可以指定要取出幾筆資料

例如取出前十筆資料

df.head(10)

head 跟 tail 也可以合併使用

譬如說

df.head(20).tail(10)

這樣就會取出前二十筆中最後十筆的資料

pandas sort_values

Posted on 2021-06-28 Edited on 2021-07-20

sort_values()

想對資料做排序的時候，可以利用 sort_values()

Sort by column

df.sort_values(by='column_name')

只要指出要對哪個 column 做排列就可以

要注意的是 sort_values()

會回傳一個排序好的 DataFrame

而不會去改變原本的排序

Sort by multiple columns

也可以同時對複數個 column 做排序

df.sort_values(by=['column1', 'column2'])

Sort Descending

另外也可以指定是昇序或是降序

預設會是昇序，所以需要降序的時候再設定就好

df.sort_values(by='column_name', ascending=False)

對複數個 column 做排序的時候

也可以分別指定昇序，或是降序

df.sort_values(by=['column1', 'column2'], ascending=[True, False])

pandas value_counts

Posted on 2021-06-25 Edited on 2021-07-20

value_counts

用來計數 Series 裡面值出現的次數

s.value_counts()

normalize

normalize 設成 True 就會回傳值出現次數佔的比例

s.value_counts(normalize=True)

dropna

dropna 設成 False 就會回傳 NaN 出現的次數

s.value_counts(dropna=False)

sort

value_counts 預設會對出現的次數做排序

可以利用 sort 設定成不要排序

s.value_counts(sort=False)

ascending

value_counts 預設會對出現的次數做排序

而排序的方式是預設從大排到小

如果想反過來可以設定 ascending 為 True

s.value_counts(ascending=True)

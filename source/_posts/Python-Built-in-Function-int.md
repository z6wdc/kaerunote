Python Built-in Function 的 [int()](https://docs.python.org/3/library/functions.html#int)

## Example 1

可以把 `str` 轉換成 `int`

```py
int('29')
```

Output

```bash
29
```

## Example 2

可以把 `float` 轉換成 `int`
轉換的時候，會直接捨去小數點後的部分

```py
int(3.14)
```

```py
int(7.0)
```

Output

```bash
3
```

```bash
7
```

isnull()

可以用 isnull() 來找出 NaN 的值

也可以寫成 isna()

df.isnull()

也可以和 any() 跟 sum() 搭配使用

sum()

可以計算每一個 column 各有幾個 NaN

df.isnull().sum()

也可以計算每一個 row 各有幾個 NaN

df.isnull().sum(axis=1)

any()

df.isnull().any()

query = df.isnull().any(axis=1)
df[query].head()

pandas groupby

Posted on 2021-06-30 Edited on 2021-07-02

groupby

groupby 用來對資料做集結

df.groupby('column1').sum()

也可對複數個 column 做集結

df.groupby(['column1','column2']).sum()

集結後也可以指定對哪個 column 做處理

df.groupby('column1').sum()['column2']

也可以指定複數個 column

df.groupby('column1').sum()[['column2','column3']]

不指定的話，就是對 column1 以外的 column 做處理

關於處理，也可以同時指定多個

例如同時使用 count sum max min

df.groupby('column1').agg(['count','sum','max','min'])['column2']

head

tail

pandas 可以利用 head 跟 tail 來取出前幾筆跟後幾筆的資料

df.head(10)

pandas sort_values

Posted on 2021-06-28 Edited on 2021-07-02

sort_values()

想對資料做排序的時候，可以利用 sort_values()

Sort by column

df.sort_values(by='column_name')

只要指出要對哪個 column 做排列就可以

要注意的是 sort_values()

會回傳一個排序好的 DataFrame

而不會去改變原本的排序

Sort by multiple columns

也可以同時對複數個 column 做排序

df.sort_values(by=['column1', 'column2'])

Sort Descending

另外也可以指定是昇序或是降序

預設會是昇序，所以需要降序的時候再設定就好

df.sort_values(by='column_name', ascending=False)

對複數個 column 做排序的時候

也可以分別指定昇序，或是降序

df.sort_values(by=['column1', 'column2'], ascending=[True, False])

Python mutable immutable

Posted on 2021-05-22 Edited on 2021-07-02

Python 的 object 有分成

mutable 跟 immutable 兩種

mutable object 是指

object 的值可以被更改

immutable 則反之

當 object 被 create 之後，值就不能被更改

mutable example

例如：list, dictionary

list example

numbers = [1, 2, 3]
numbers[2] = 4
print(numbers)

Output

[1, 2, 4]

immutable example

例如：numbers, string, tuple

但 immutable object 還需要注意

值不能被更改的意思

numbers example

a = 3
print(id(a))
a = 4
print(a)
print(id(a))

Output

4

可以看到 a 的值被更改了

但如果用 id() 把 a 的 address 印出來的話

就會發現 a 的 address 已經改變了

因為 a 是 immutable object

所以對 a 指定新的值的時候

就會被指去新的 address

雖然值更換了，但也不是原本的 object 了

tuple example

還有另外一個關於 tuple 的例子

雖然 tuple 是 immutable object

但如果 tuple 裡面包含 mutable object

譬如說 list

那在 tuple 裡面的 list 是可以更改值的


t = (1, [1, 2, 3])
t[1][2] = 4
print(t)

Output

(1, [1, 2, 4])

但如果想把 tuple 裡面的 list 變成 numbers 的話

t = (1, [1, 2, 3])
t[1] = 2

就會跳出下面的 error

TypeError: 'tuple' object does not support item assignment

Python strip

Posted on 2021-07-15 Edited on 2021-07-18

strip()

strip() 可以用來刪除字串前後的字元

'www.example.com'.strip('cmowz.')
# 'example'

也可以用來刪除空白或是換行記號

預設值是刪除空白所以不用輸入參數

' example '.strip()
# 'example'
'\nexample\n'.strip('\n')
# 'example'

lstrip()

lstrip()

lstrip() 只會刪除左邊的字元

'www.example.com'.lstrip('cmowz.')
# 'example.com'

rstrip()

rstrip()

rstrip() 只會刪除右邊的字元

'mississippi'.rstrip('ipz')
# 'mississ'

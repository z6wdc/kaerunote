git config core.ignorecase

Posted on 2021-07-15 Edited on 2021-07-20

core.ignoreCase

雖然預設為 false

但如果用了 git clone 或是 git init

就會設定成 true

如果需要可以再度更改為 false

git config core.ignorecase false

如果想直接把檔案名稱改成大寫的話

也可以利用下面的 command

git mv hello.txt Hello.txt

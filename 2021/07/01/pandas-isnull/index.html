pandas isnull

Posted on 2021-07-01 Edited on 2021-06-30

isnull()

可以用 isnull() 來找出 NaN 的值

也可以寫成 isna()

df.isnull()

也可以和 any() 跟 sum() 搭配使用

sum()

可以計算每一個 column 各有幾個 NaN

df.isnull().sum()

也可以計算每一個 row 各有幾個 NaN

df.isnull().sum(axis=1)

any()

用 any() 來判斷每一個 column 有沒有 NaN 值

df.isnull().any()

想要看那一個 row 缺值的話

query = df.isnull().any(axis=1)
df[query].head()

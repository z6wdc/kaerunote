pandas shape

Posted on 2021-06-24

想知道 DataFrame 有幾個 row 跟 columns 的時候

可以使用 shape

df.shape

如果只想知道 row 或是 column 分別個數時

後頭加上 index 0 或 1

就可以分別取得 row 跟 column 的個數

row

df.shape[0]

column

df.shape[1]
